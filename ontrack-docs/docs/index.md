# Welcome to Yontrack {{ yontrack_version }}

Yontrack goals are about simplifying the delivery process, giving quick and high-level insights, bringing transparency and confidence to teams.

It does this by collecting everything that happens in your CI/CD ecosystem, bringing powerful visualizations, providing feedback and ways to speed up your delivery by using the collected data for further automation.
